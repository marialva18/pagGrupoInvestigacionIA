# Roles y permisos

## EDITOR

- Crea sus contenidos o trabaja en los asignados.
- Edita borradores.
- Adjunta imágenes.
- Usa autoguardado.
- Envía a revisión.
- Responde comentarios.
- Edita secciones institucionales autorizadas.

## ADMIN

Incluye las capacidades editoriales y además:

- Publica y archiva.
- Gestiona cuentas, permisos y sesiones.
- Promueve o degrada usuarios.
- Configura fuentes, miembros y datos institucionales.
- Consulta analítica y auditoría.

## Reglas

- Sin registro público.
- No puede quedar el sistema sin un administrador activo.
- Cada cuenta identifica a una persona.
- Los cambios de rol revocan sesiones y generan auditoría.
